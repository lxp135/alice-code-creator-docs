# Alice CodeCreator 中文参考文档
---

> [Alice CodeCreator](http://creator.liuxp.me) 是一个通用代码生成辅助工具，支持生成包括不限于html、js、java、xml、sql等各种类型代码。

该文档包含三个主要组成部分:

* **用户手册** - 当您仅使用Alice CodeCreator生成代码时参考该部分文档
* **开发者手册** - 当您想扩展Alice CodeCreator功能时可以参考该部分文档
* **参考资料** - 一些相关的文档与资料
